# Gemini in Google Docs

Gemini in Google Docs acts as your intelligent co-author. Whether you are starting with a blank page or refining an existing draft, Gemini can help.

## Help Me Write

When you open a new document, you'll see the **Help me write** prompt. 

Use the 4-step framework (Persona, Task, Context, Format) to generate the first draft of your document.

1. Click the **Help me write** button.
2. Enter your prompt.
3. Click **Create**.
4. You can ask Gemini to refine the draft: *Make it more formal*, *Shorten it*, or *Elaborate*.

## AI Building Blocks

If you need to create structured content quickly, you can use AI building blocks within your documents.

Type `@` anywhere in your document to access AI building blocks.

<Card title="Available Building Blocks" icon="cube">
- **Generative Tables**: Ask Gemini to create a table. *Example: "@table of project roles and responsibilities."*
- **Drafting Emails**: Draft an email directly in Docs to collaborate with others before sending it via Gmail. *Type `@email draft`.*
- **Custom Variables**: Use dropdowns and pill markers to create reusable templates.
</Card>
